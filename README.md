# GeolocationEngineering

The Geolocation-Engeneering Toolkit is to obtain the exact location of a user using an fake maps page through ngrok fowarding.

# Usage:
First Unzip ngrok:
```
unzip ngrok.zip
```
```sh
chmod +x tracker.sh

sudo ./tracker.sh
```
